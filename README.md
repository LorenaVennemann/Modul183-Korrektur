f. Um eine Nachricht entschlüsseln zu können die via asymmetrischer Verschlüsselung versendet wurde, ist der Private Key des Empfängers sowie manchmal der Public Key des Senders nötig (für die Identitätsprüfung des Absenders).
✅ Richtig

**Begründung:** 
Bei der asymmetrischen Verschlüsselung wird die Nachricht mit dem öffentlichen Schlüssel (Public Key) des Empfängers verschlüsselt. Nur der private Schlüssel (Private Key) des Empfängers kann sie wieder entschlüsseln.

Wenn man zusätzlich prüfen will, ob die Nachricht wirklich vom Absender kommt, kann der Absender sie mit seinem privaten Schlüssel unterschreiben. Der Empfänger kann diese Unterschrift dann mit dem öffentlichen Schlüssel des Absenders überprüfen.

So bleibt die Nachricht geheim und man weiß auch, von wem sie ist.
